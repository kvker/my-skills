# Expand to Article

将笔记内容扩写为专业技术文章，直接覆盖原文件。

## 输入

用户会提供：文件路径（绝对路径或相对于当前目录的路径）

## 工作流程

### 1. 读取文件

读取用户指定的文件，获取完整内容。如果文件不存在或无法读取，告知用户错误。

### 2. 分析内容

分析原文，判断：
- 文章主题/标题
- 技术领域（前端、后端、AI、DevOps 等）
- 现有结构（是否有标题层级、代码块、列表等）
- 内容深度（概念解释、教程、经验分享、还是笔记片段）

### 3. 添加 Frontmatter

在文件顶部添加 YAML frontmatter：

```yaml
---
title: 文章标题
date: YYYY-MM-DD
tags: [tag1, tag2, tag3]
description: 一句话描述文章内容
---
```

规则：
- `title`：从内容中提取或推断，无标题则基于主题生成
- `date`：当前日期（YYYY-MM-DD 格式）
- `tags`：根据内容推断技术标签，1-4 个为宜，标签内不带空格，用 `-` 连接多词标签（如 `ai-tools`）
- `description`：一句话概括核心内容，不超过 100 字

### 4. 扩写内容

保持原文核心信息，扩展为结构清晰的技术文章。

**结构**：正文标题从 H2 开始，frontmatter 中的 title 作为文章主标题。

**风格**：
- 简短专业，口吻直接
- 不使用"当今"、"现如今"等号召类开头
- 描述完内容即结束，不留 Q&A、关注引导等
- 根据原文方向调整：概念类解释原理、教程类步骤清晰、笔记片段整理归类、经验分享结构化表述

**要求**：
- 不改变原文核心观点
- 扩写而非改写，保持作者原意
- 删除口语化表达，改为书面语
- 精炼表达，讲清楚即可

### 5. 写回文件

将处理后的完整内容（frontmatter + 正文）覆盖写入原文件。

## 示例

**输入**：
```markdown
Claude Code 是个 AI 编程工具
可以用自然语言写代码
支持多文件编辑
```

**输出**：
```yaml
---
title: Claude Code AI 编程工具简介
date: 2026-04-22
tags: [AI,Claude,dev-tools]
description: 介绍 Claude Code 的基本功能和使用场景。
---

Claude Code 是 Anthropic 推出的 AI 编程助手，通过自然语言与开发者交互，完成代码编写、编辑和多文件协作等任务。

## 核心功能

- **自然语言编程**：用日常语言描述需求，AI 自动生成代码
- **多文件编辑**：支持跨文件重构和优化
- **上下文理解**：理解项目结构和代码上下文，生成更准确的建议

## 使用场景

1. 快速原型开发
2. 代码重构和优化
3. 学习和探索新框架
```

## 错误处理

- 文件不存在：告知用户
- 文件为空：告知用户
- 读取失败：提示检查权限
